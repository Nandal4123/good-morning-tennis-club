## 🧪 다음 테스트 항목

### 1. 관리자 비밀번호 설정 테스트
- [ ] "관리자 비밀번호" → "설정" 버튼 클릭
- [ ] 모달 표시 확인
- [ ] 비밀번호 입력 → 저장
- [ ] 저장 후 상태 변경 확인 ("비밀번호가 설정되어 있습니다.")

### 2. 가입 코드 설정 테스트
- [ ] "가입 코드" → "설정" 버튼 클릭
- [ ] 모달 표시 확인
- [ ] 가입 코드 입력 → 저장
- [ ] 저장 후 상태 변경 확인 ("가입 코드가 설정되어 있습니다.")

### 3. 클럽 추가 테스트
- [ ] "클럽 추가" 버튼 클릭
- [ ] 모달 표시 확인
- [ ] 새 클럽 생성 시도
  - 클럽 이름: "Test Club"
  - 서브도메인: "test-club"
  - 관리자 비밀번호: "test1234" (선택)
  - 가입 코드: "test5678" (선택)
- [ ] 클럽 생성 성공 확인
- [ ] 목록에 새 클럽 추가 확인

### 4. 클럽 이동 테스트
- [ ] "이동" 버튼 클릭 → 해당 클럽으로 이동 확인
- [ ] "새 탭" 버튼 클릭 → 새 탭에서 클럽 열기 확인

### 5. Good Morning Club 선택 테스트
- [ ] Good Morning Club 클릭
- [ ] 상세 정보 표시 확인
- [ ] 통계 정보 확인
